# TOC and HOC Decarbonization Potential (sample)

**Legend.** TOC = Transport Operation Category; HOC = Hub Operation Category. CO2e intensity values are kg CO2e per tonne-km (WTW = Well-to-Wheel, TTW = Tank-to-Wheel) as reported by the iLEAP public demo server. **Decarbonization potential is an indicative, illustrative figure** derived from this small sample: the percentage WTW-intensity reduction an entry could reach by matching the lowest-intensity option of the same type in the sample. It is not advice and not a real abatement estimate.
